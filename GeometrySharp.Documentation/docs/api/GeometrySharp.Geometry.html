NurbsCurve

A NURBS curve - this class represents the base class of many curve types and provides tools for analysis and evaluation. This object is deliberately constrained to be immutable. The methods deliberately return copies. ///

NurbsSurface

A simple data structure representing a NURBS surface. NurbsSurfaceData does no checks for legality. You can use for that.

Plane

Polygon

A closed planar Polyline.

Polyline

A simple data structure representing a polyline. PolylineData is useful, for example, as the result of a curve tessellation.

Ray

A Ray is simply an position point and normal.

Triangle

A triangular face of a mesh.

Vector3

Vector3 is represented simply by an list of double point numbers. So, you would write simply [1,0,0] to create a Vector3 in the X direction.

Volume

A simple data structure representing a NURBS volume. This data structure is largely experimental in intent. Like CurveData and Surface, this data structure does no legality checks.
